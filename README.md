# Inflation Calculator

A modern **React** application for analyzing inflation trends and calculating how the value of money changes over time. Supports multiple countries, percent or money calculation, and interactive charts.

---

## Features

- Calculate inflation in **percent** or **money value**.
- Supports multiple countries (Ukraine, Denmark, etc.).
- Interactive and responsive charts using **Recharts**.
- Clean, modern UI with **TailwindCSS**.
- Mobile-friendly design.
- Hero section with brief app description.

## Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/inflation-calculator.git
cd inflation-calculator
```
Install dependencies:

2. npm install

```bash
npm install
```

3. Start the development server:

```bash
npm start
```

4. 
```bash
Open http://localhost:3000
 in your browser.
 ```

---

## Usage

1. Enter an amount of money.
2. Choose a start date and end date.
3. Select a country.
4. Choose the calculation type:
    - Inflation percent
    - Money value
5. Click Calculate.
6. View results and interactive chart.


## Customize

- Add more countries in InflationForm.jsx.
- Adjust chart styling in InflationChart.jsx.
- Use dark mode by adding TailwindCSS dark classes.

## Dependencies

- React
- TailwindCSS
- Recharts
- Chart.js (optional, legacy)
- Other standard React dependencies
